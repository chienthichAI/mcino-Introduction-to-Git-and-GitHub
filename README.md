## Simple Interest Calculator

This is a simple bash script to calculate simple interest.

### Input
The script accepts the following inputs:
*   **p**, principal amount
*   **t**, time period in years
*   **r**, annual rate of interest

### Output
The script outputs the simple interest calculated using the formula:
`simple interest = p * t * r / 100`

### Usage
To run the script:
1.  Clone the repository.
2.  Run `bash simple-interest.sh`.
3.  Enter the values when prompted.
